# Resep Dunia

Resep Dunia adalah aplikasi Android sederhana yang menampilkan katalog makanan dessert dari API TheMealDB. Aplikasi ini dibuat menggunakan Java dan Android Studio.

## Fitur Aplikasi

- Menampilkan daftar makanan dessert
- Menampilkan gambar makanan
- Menampilkan detail resep makanan
- Mengambil data dari API online
- Mendukung refresh data
- Menampilkan data menggunakan RecyclerView berbentuk grid

## Teknologi yang Digunakan

- Java
- Android Studio
- RecyclerView
- View Binding
- Volley
- Glide
- SwipeRefreshLayout
- TheMealDB API

## API yang Digunakan

Daftar dessert:

```text
https://www.themealdb.com/api/json/v1/1/filter.php?c=Dessert
